# Student Grade Calculator

## Description
The Student Grade Calculator is a Java application that calculates a student's total marks, average percentage, and grade based on the marks entered for each subject.

The program accepts marks for multiple subjects, calculates the overall percentage, and assigns a grade according to the student's performance.

## Features
- Accepts marks for any number of subjects
- Calculates total marks obtained
- Calculates average percentage
- Assigns grades automatically
- Displays final result in a clear format
- Simple and user-friendly interface

## Technologies Used
- Java
- Scanner Class

## Grade Criteria

| Percentage | Grade |
|------------|--------|
| 90% and above | A |
| 80% - 89% | B |
| 70% - 79% | C |
| 60% - 69% | D |
| 40% - 59% | E |
| Below 40% | F |
